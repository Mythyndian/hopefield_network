# hopefield_network

Implementation of Hopefield Network.

## Description

Implementation of Hopefield Network with GUI for easier way to input long vectors.

## Getting Started

### Dependencies

* Python
* Numpy
* Tkinter
* Windows/Linux

### Executing program

* How to run the program
```
python main.py
```
